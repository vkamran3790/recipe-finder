🟢 JavaScript Methods
🔥 1. abort()

📌 ساختار:

searchController.abort();

💡 معنی:

درخواست fetch که هنوز تمام نشده را لغو می‌کند.

مثلاً:

کاربر سرچ می‌کند:

pizza

درخواست ارسال می‌شود:

API → pizza

اما قبل از اینکه جواب بیاید، کاربر می‌نویسد:

chicken

پس درخواست قبلی دیگر لازم نیست.

با:

abort()

درخواست قبلی را متوقف می‌کنیم.

🎯 کاربرد:

جلوگیری از ارسال درخواست‌های اضافی.

مثلاً در:

Search
Autocomplete
Live Search
🔥 2. AbortController()

📌 ساختار:

let searchController = new AbortController();

💡 معنی:

یک کنترل‌کننده برای مدیریت یک عملیات قابل لغو می‌سازد.

یعنی به ما یک دکمه Stop برای fetch می‌دهد.

مثال:

searchController = new AbortController();

اینجا یک کنترلر جدید ساختیم.

بعد:

fetch(url,{
    signal: searchController.signal
})

یعنی:

این fetch را به این کنترلر وصل کن.

اگر بعداً:

searchController.abort();

اجرا شد:

این fetch متوقف شود.

🔥 3. signal

📌 ساختار:

signal: searchController.signal

💡 معنی:

رابط بین:

AbortController
        |
        |
      signal
        |
        |
      fetch

است.

یعنی:

"اگر controller گفت stop، این fetch هم متوقف شود."

🔥 4. error.name

📌 ساختار:

error.name

💡 معنی:

اسم نوع خطایی که اتفاق افتاده را می‌دهد.

مثلاً:

catch(error){

console.log(error.name);

}

ممکن است خروجی:

AbortError

باشد.

یعنی:

خطا به خاطر abort اتفاق افتاده.

مثال:

if(error.name === "AbortError"){
    return;
}

یعنی:

اگر فقط درخواست را خودمان لغو کردیم، کاری نکن.

🟢 Array Methods
🔥 13. slice()

📌 ساختار:

array.slice(start,end)

💡 معنی:

یک بخش از Array را کپی می‌کند.

مثال:

"Pizza",
"Burger",
"Pasta",
"Rice"
]


foods.slice(0,2)

خروجی:

[
"Pizza",
"Burger"
]

⚠️ Array اصلی تغییر نمی‌کند.

🎯 کاربرد:

Load More

مثلاً:

recipes.slice(0,6)

یعنی:

فقط 6 رسپی اول را نشان بده.
